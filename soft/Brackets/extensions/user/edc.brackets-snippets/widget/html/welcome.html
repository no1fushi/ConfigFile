What's new?

  • 1.8.1
  • Bug fixes
  • 1.8.0
  • A new `Any` scope option for global snippets (#38)
  • Follow whatever indentation settings you've set in Brackets preferences (#20 / #28 / #40)
  • New snippet editor (Ace is awesome)
  • New php snippets in library
  • Customizable keystroke (Only `Select next variable` so far)
  • Confirmation on unsave (#21)
  • 1.7.x
  • Support multiple snippets in the same line! (Fix #15)
  • Support same orders of variables and cursors, eg: console.${1:log}(${1})
  • Support files with multiple syntaxes, like erb/ejs
  • Improved UI on dark theme
  • 1.6.0 Please read this if you are updated from 1.5.0 and below
  • COOL Tab indentation support
  • COOL Import and Export
  • COOL Show snippets according to code context rather than file type (eg: <style> in HTML)
  • Differentiate between the pre-defined and the user-defined snippets
  • Judge if user tends to indent line or go to next placeholders when Tabbed
  • Recongize trigger right after >
  • New PHP snippets in library (thanks erknrio)
